


# VK Ads Cases Parser

Этот проект предназначен для парсинга кейсов по таргетированной рекламе с сайта [VK Реклама](https://ads.vk.com/cases). Скрипт собирает информацию о каждом кейсе: заголовок, ссылку и дату публикации.

## Установка

1. Клонируйте репозиторий:


```bash
git clone https://github.com/ntstnd/parsing.git
cd parsing
````

2. Рекомендуется создать виртуальное окружение:

```bash
python -m venv venv
venv/Scripts/activate  # для Windows
```

3. Установите зависимости:

```bash
pip install -r requirements.txt
```
## Используемые библиотеки
Проект использует BeautifulSoup4, requests.

## Использование

В проекте есть скрипт, который считывает HTML-файл с кейсами и парсит нужные данные. 
Скрипт выводит на экран список кейсов в формате:

```
--- Result 1 ---
заголовок: Пример заголовка
ссылка: https://ads.vk.com/cases/...
дата публикации: 01.01.2025
```

## Лицензия

Проект предоставлен "как есть" и может быть использован для учебных целей.
